
## AI 旅行攻略生成器 Web 应用

### 完成内容
- 基于 TripStar Agent 架构分析，设计并实现了旅游攻略生成 Web 应用
- 三层架构：FastAPI 后端 + LLM 编排 + 文档生成
- 前端：纯 HTML/CSS/JS，对话式输入，SSE 流式输出
- 支持 HTML/PDF/DOCX 三格式下载

### 技术栈
- Python 3.13 + FastAPI + DeepSeek API
- Jinja2 模板渲染
- WeasyPrint PDF (需系统依赖) + python-docx DOCX
- 无登录、无数据库、纯内存缓存

### 关键文件
- app.py: FastAPI 入口
- orchestrator.py: AI 编排层
- prompts.py: System Prompt
- generator.py: 文档生成 (HTML/PDF/DOCX)
- static/: 前端页面
- templates/guide.html: 攻略展示模板

### 待用户配置
- 在 .env 中填入 DeepSeek API Key
- PDF 需要 brew install pango cairo glib

## 修复：每日行程后内容丢失

### 根因
SSE 传输中 LLM 输出的 newline 字符被吞掉，导致 markdown 没有段落分隔。
后端解析器把全部内容当成一个 h1 块，h2/h3/table 全部未识别。
所以 分日行程之后的所有 section 都不输出。

### 修复
1. orchestrator.py: 不再逐字符发送 content 事件，改为缓冲 200 字符或遇到换行再发
2. app.py: SSE 数据含 \n 时使用标准多行格式（多个 data: 行）

### 验证
- 3 日游: 9/9 sections 全部输出 ✅
- 15 日游: 8/9 sections 输出（知识图谱缺失，LLM 长文本截断已知限制）
